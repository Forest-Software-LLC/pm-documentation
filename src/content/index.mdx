# Introduction

Forest Package Manager (Forest PM) is a cloud-based code package manager designed for immersive experience developers.

# Why Forest?

### A-Z package management

Forest handles uploading, maintaining and distribution of your public and private packages out of the box.

### Designed for teams and individuals

Whether you're a solo dev, or part of a large studio, forest has the tools to enhance your workflow at scale.

### Beginner Friendly

Forest PM is designed to be approachable for beginner devs and professionals alike. Never before has an industry leading workflow been more accessable.
